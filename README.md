## 구현기능

### 사용자

**로그인/회원가입**

- 로그인
- 회원가입
- ID/PW 찾기

**자료 찾기**

- 통합검색
- 신착자료
- 베스트자료

**신청/참여**

- 희망도서 신청
- 멀티미디어 좌석예약
- QNA

**내서재**

- 대출 조회/연기
- 대출이력
- 예약도서 조회
- 희망도서 신청내역
- 좌석 예약 조회
- 관심도서 조회
- 회원정보 수정

### 관리자
  
**자료관리**

- 희망도서 등록
- 자료등록
- 자료관리

**대출관리**

- 대출/반납/연체 관리

**회원관리**

- 회원관리

**도서관 관리**

- 멀티미디어 좌석 조회
- FAQ 조회

**통계관리**

- 대출/반납 통계
- 도서 보유 현황
- 이용자 현황
